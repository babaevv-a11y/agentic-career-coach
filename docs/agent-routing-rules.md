Agent Routing Rules

Lead Orchestrator answers directly when:
- the request is general, informational, or planning-based
- no current job data or pipeline data is needed

Lead Orchestrator delegates to Career Specialist when:
- job search is requested
- internship pipeline data is needed
- status changes are needed
- saved records, notes, or deadlines must be reviewed

Career Specialist uses:
- fetch_jobs for internship search
- sync_pipeline for create/list/update pipeline work
